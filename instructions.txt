CVM++ — Build & Run Instructions
==================================

REQUIREMENTS
------------
  C++17 compiler (GCC 9+ or Clang 10+ or MSVC 2019+)
  CMake 3.16+

BUILD
-----
  cmake -B build
  cmake --build build

  The compiled binary is:  build/cvm

RUNNING A SOURCE FILE (.cvm)
-----------------------------
  ./build/cvm samples/fizzbuzz.cvm

  With debug flags:
  ./build/cvm --dump-ast       samples/fizzbuzz.cvm   # show AST
  ./build/cvm --dump-bytecode  samples/fizzbuzz.cvm   # show bytecode
  ./build/cvm --trace          samples/fizzbuzz.cvm   # trace every instruction
  ./build/cvm --profile        samples/fizzbuzz.cvm   # execution profile

COMPILE TO BYTECODE (.cvmb) AND RUN SEPARATELY
-----------------------------------------------
  Step 1 — compile:
    ./build/cvm --compile-only samples/calculator.cvm
    # produces: samples/calculator.cvmb

  Step 2 — run bytecode:
    ./build/cvm --run samples/calculator.cvmb

  Or run a .cvmb directly by extension:
    ./build/cvm samples/calculator.cvmb

INTERACTIVE REPL
----------------
  ./build/cvm

  REPL toggle commands:
    :ast        — show AST for each input
    :bytecode   — show bytecode for each input
    :trace      — step-by-step VM execution
    :optimize   — toggle constant folding
    :clear      — reset global variables
    :quit / :q  — exit

SAMPLE PROGRAMS
---------------
  echo "15"  | ./build/cvm samples/fizzbuzz.cvm
  echo "0"   | ./build/cvm samples/truth_machine.cvm
  echo "1"   | ./build/cvm samples/truth_machine.cvm
               ./build/cvm samples/functions_demo.cvm
               ./build/cvm samples/strings_demo.cvm
               ./build/cvm samples/calculator.cvm

  Scene 1 — Show the source code:
    cat samples/truth_machine.cvm

  Scene 2 — Show the parsed AST (no input needed):
    ./build/cvm --dump-ast samples/functions_demo.cvm

  Scene 3 — Show the compiled bytecode (no input needed):
    ./build/cvm --dump-bytecode samples/functions_demo.cvm

  Scene 4 — Compile to a .cvmb binary file:
    ./build/cvm --compile-only samples/truth_machine.cvm
    ls -lh samples/truth_machine.cvmb

  Scene 5 — Run the .cvmb file directly (no source code needed):
    ./build/cvm --run samples/truth_machine.cvmb
    (enter 0 — prints 0 and exits)
    ./build/cvm --run samples/truth_machine.cvmb
    (enter 1 — prints 1 forever, press Ctrl+C to stop)

  Scene 6 — Show the execution profiler:
    echo "20" | ./build/cvm --profile samples/fizzbuzz.cvm
